# 我的个人博客

这是一个基于 Jekyll 构建的个人博客，部署在 GitHub Pages 上。

## 特性

- 🎨 现代化的响应式设计
- 📱 移动端友好
- 🏠 固定侧边栏，独立滚动
- 📸 瀑布流相册布局
- 📝 Markdown 文章支持
- 🔍 搜索功能
- 🏷️ 分类和标签系统
- 💡 灯箱图片查看
- 🚀 快速加载

## 技术栈

- **Jekyll** - 静态网站生成器
- **Tailwind CSS** - CSS 框架
- **JavaScript** - 交互功能
- **Markdown** - 内容编写

## 本地运行

### 环境要求

- Ruby 2.7+
- Bundler

### 安装依赖

```bash
bundle install
```

### 启动本地服务器

```bash
bundle exec jekyll serve
```

访问 `http://localhost:4000` 查看网站。

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 等待构建完成
